# Hotel Recommendation Dashboard

## 🚀 Quick Start Guide

### 1. Backend Setup
```bash
cd backend
pip install -r requirements.txt
# Configure your .env file with Neo4j credentials
python init_db.py
python main.py
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
